===================================================
= INSTRUCTIONS FOR MASTER SHEET USE CONSIDERATION =
===================================================

There are pros and cons to everything, and master sheet usage is no different. Still, I highly recommend
their use especially as it will reduce your file count overhead in the long run. However, if you are using
a Paper Doll system like this tileset is honestly built for, you'll need to copy such a design change across
your entire system. This means using master sheets for the base sprites, tools, etc. So that is one such
con.  The other con is large file sizes. You would need a roughly 2048 x 2048 sprite sheet for each base, eye,
etc. These dimensions mean larger file sizes which creates more memory load overhead, in simple terms, takes a
longer time to load into memory. However, this con is offset by the speed and efficiency of modern technology
and should not pose a problem for you, especially if you are using memory wisely with your game engine.

= MANA SEED EYES HAS BEEN FINALIZED =
=====================================
Recently, Seliel the Shaper indicated he plans to move in a different direction from the base asset and
their weapon (BOW, ONE, POL) counterparts. As such, we should not expect many more updates on these
assets going forward, and thus there should be no reason to further update Mana Seed Eyes. So unless that
changes or people find bugs in the system, this will likely be the final version.
